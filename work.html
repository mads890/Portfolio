Websites and Applications

Adventure Time Quiz

This interative web application is a game in which the user must answer questions in order to rescue the princesses of Ooo, the fictional kingdom of the cartoon's universe. Upon submission, users are provided with textual and graphic feedback as well as an interactive element (a button) that allows them to move on to the next question. At the end of the quiz, there are three possible outcomes a user may achieve depending on the number of questions they got right.

This project uses HTML, CSS, JavaScript, and jQuery.

Check out my repository on GitHub, or try the live app yourself!
